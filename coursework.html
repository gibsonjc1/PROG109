Spring 2018

PC Analysis & Configuration I/II

  • TECH 215/217
  • 10 Credits
  • Instructor: Rudy Helm
  • Presents the basics of PC hardware and software installation and configuration. Students gain hands-on instruction. Course includes about 50% of the competencies required for A+ certification.
    Topics include:
    • BIOS & Motherboards
    • Storage
    • System optimization
    • Printers
    • Virtualization
    • Mobile computing
    • Resource management
    • Installation, configuration, and optimization of operating systems.
    • Introductory network installation and configuration.
    • Basic electrical concepts and measurement.
    • Advanced troubleshooting techniques.
  • I've had a large amount of PC building and repair experience in my life, so going into the class I was expecting it to be almost entirely review for me. That said I OK without having the option to test out of it as I figured it would be a nice class to take as my first college class in about 15 years, especially while working full time as my employment contract wound down. In the end, while I did learn some entirely new information, I was surprised with how much of the material I realized that I used to know but had forgotten from lack of use.

Summer 2018 (Current Quarter)

Introduction to Web Development

  • PROG 109
  • 5 Credits
  • Instructor: Maricel Medina-Mora
  • Introduces fundamental programming techniques using current web development software. Students design procedures and write computer instructions to solve business problems, learn procedural programming, develop graphical user interfaces for the web and work with events and objects.
  • While I had done some simple programming in high school, I hadn’t done any in a complex language, so I was expecting this to be the class where I would need to learn the most this summer. While I was right about how much I needed to learn, I was surprised at how easily I learned HTML and CSS (the only part that gave me a challenge was floating elements), but how much I struggled with JavaScript, especially DOM model events and passing information back and forth between the HTML page and the JavaScript. Once the data was within the JavaScript though, I found it my prior experience with loops and if statements really helped me, and I was able to find and fix bugs quickly.

Calculus I

  • MATH& 151
  • 5 Credits
  • Instructor: Ryan Bauer
  • Introduces the concepts of limits, derivatives, and integrals. Topics include techniques and applications of derivatives of algebraic and transcendental functions. Students begin working with anti-derivatives.
  • After not having taken a math class in over 5 years, Bellevue College required me to retake math. Previously I had taken math though Calculus II, that was over 20 years ago so while I expected a lot of review, I didn’t really expect to learn anything new. While this was mostly correct I was surprised to learn some new nuances of derivates, like Rolle’s Theorem, l’Hospital’s Rule, and how limits can be used with derivatives, as opposed how I thought that limits stopped being very useful once derivatives were introduced.

Information Security Essentials

  • IT 128
  • 5 Credits
  • Instructor: Ahmed Saleh
  • Introduces concepts and issues related to securing information systems and the development of policies to implement information security controls. Topics include security vulnerabilities, threats, defense measures, and the legal and ethical issues associated with information security. Students will learn how to apply security best practices in multiple security architectures.
  • I really didn’t know what to expect from this class, as I felt like I had a handle on how to secure computes in the home environment, but I had no real knowledge of the security in the medium business environments, let alone enterprise environments. As the class has progressed, I feel like I’m starting to understand most of the overriding principles that go into securing a large network, but I’d a little disappointed that there’s very little practical hands on knowledge being taught.

Technical Writing

Emphasis on audience analysis, clear and effective writing style, and use of visual elements, by creating documents in a variety of professional report formats, such as memos, proposals, progress reports, completion reports, and instruction manuals.
